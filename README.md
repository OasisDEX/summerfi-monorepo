# Summerfi monorepo

## Structure

### Apps

- `summerfi-api`: Summerfi API

### Packages

- `sdk`: Summerfi SDK
- `eslint-config-custom`: `eslint` configurations (includes `eslint-config-next` and
  `eslint-config-prettier`)
- `tsconfig`: `tsconfig.json`s used throughout the monorepo

## Commands

### Build

To build all apps and packages, run the following command:

```shell
pnpm build
```

### Develop

To develop all apps and packages, run the following command:

```shell
pnpm dev
```

## Submodules

To initialize submodules use:

`git submodule add -f git@github.com:OasisDEX/summer-earn-protocol.git earn-protocol`
